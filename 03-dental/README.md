# 歯科医院 テンプレ

このディレクトリは「歯科医院」業種向けの HP テンプレートです。
**サンプル歯科医院** のデモLPをベースに制作しています。

## このテンプレの特徴

詳細はルートの [README.md](../README.md) を参照してください。

## クイックスタート

```bash
npm install
npm run dev   # http://localhost:3000
```

## カスタマイズ前にやること

```bash
# 固有名詞を grep で洗い出し
grep -rE "サンプル歯科医院" src/

# 必要に応じて一括置換
```

固有名詞の置換が終わったら、画像（`public/images/`）も業種に合わせて差し替えるか、ChatGPTで再生成してください。
