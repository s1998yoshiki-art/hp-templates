# HP テンプレート集（5業種）｜Yoshiki Web Studio 伴走サポート受講者向け

このリポジトリは、Yoshiki Web Studio の「HP制作 1ヶ月伴走サポート」受講者向けに配布している **業種別テンプレ集** です。

各テンプレは、私（よしき）が実際にデモLPとして制作したものをベースに、業種ごとに使いやすい形にしています。**フォークして自由にカスタマイズしてください**。

## 使い方

### 1. フォーク or クローン

```bash
git clone https://github.com/[your-username]/hp-templates.git
cd hp-templates/01-koumuten  # 使う業種のディレクトリへ
```

### 2. 依存関係のインストール

```bash
npm install
```

### 3. ローカルで起動

```bash
npm run dev
```

→ http://localhost:3000 で確認

### 4. 顧客情報の置換（重要）

各テンプレには、私が制作したデモのお客様情報がそのまま残っています。
**必ず以下の手順で自社/顧客情報に置換してください**：

```bash
# 固有名詞を grep で洗い出し
grep -rE "カネヨシ|宇栄原|098-859-0395" src/

# 置換例（macOS の sed）
sed -i '' 's/カネヨシ工務店/[新しい店名]/g' src/app/page.tsx
sed -i '' 's/098-859-0395/[新しい電話番号]/g' src/app/page.tsx
```

または、お好みのエディタで一括置換してください。

### 5. 画像の差し替え

`public/images/` には私が ChatGPT で生成したサンプル画像が入っています。
ChatGPT で業種別に画像を作り直すか、お客様の実写写真に差し替えてください。

### 6. ビルド & デプロイ

```bash
npm run build
```

Cloudflare Pages にデプロイする場合：

```bash
npm run pages:deploy
```

## カスタマイズの最低ライン（重要）

テンプレを使い回す時、最低でも以下は **必ず案件ごとに差し替え** てください。
そうしないと、「悪徳業者と同じテンプレ感」が出てしまいます。

- [ ] 配色（メインカラーを1色変えるだけでも印象が変わる）
- [ ] キャッチコピー（業種ではなく、**その会社の強み** に焦点）
- [ ] 写真（実在の店舗写真 or AI 生成の業種別画像）
- [ ] お客様の声（実際の口コミから抽出）
- [ ] 実績（あれば必ず）

「**コア骨格 80% + 表面カスタム 20%**」が黄金比です。
