## クイックスタート

```bash

```

## カスタマイズ前にやること

```bash
# 固有名詞を grep で洗い出し
grep -rE "長山行政書士事務所" src/

# 必要に応じて一括置換
```

固有名詞の置換が終わったら、画像（`public/images/`）も業種に合わせて差し替えるか、ChatGPTで再生成してください。
